# EFDNSR-PyTorch

## Dependencies
* Python 3.6
* PyTorch >= 1.0.0
* numpy
* skimage
* **imageio**
* matplotlib
* tqdm
* cv2 >= 3.xx 

## How to train

We used DIV2K and FFHQ dataset to train our model. 

The model files are uploaded! You can use the EDSR framework to train our RFDN
